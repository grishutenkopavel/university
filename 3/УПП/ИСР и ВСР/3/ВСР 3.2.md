### фрагмент ТЗ "Калькулятор"
## 1. Назначение разработки

Приложение "Калькулятор" предназначено для проведения арифметических расчетов на электронно-вычислительная машине.

Приложение "Калькулятор" создается с целью обеспечить пользователя возможностью автоматизировать процесс вычисления математического выражения по заданному зарание правилу.

Приложение предназначено для использования на устройствах клиентов. Приложение доступно всем желающим его использовать.
## 2. Содержание программы
Приложение "Калькулятор" автоматизирует процесс вычисления математичесих выражений таких как: сумма, разность, произведение, поиск частного, операции с тригонометрическими функциями, корнем, логарифмами и степенью.
## 3.  Требования к программе.
Программа предствляет собой приложение с графическим интерфейсом пользователя. Основнми компонентами приложения являются: дисплей, задача которого отражать текущее состояние программы; кнопка получения результата; кнопки реализующие ввод в систему чисел; кнопки, реализующие выбор операции, которую нужно провести с числами; кнопки, реализующие использование логарифмических и тригонометрических функций.
Программа должна предоставлять свободу в порядке использования операций.
Программа должна предоставить возможность выбора приоретета операций.
Программа должна обладать способом сброса текущего состояния к состоянию по умолчанию.

...

 * Добавление кнопок с цифрами на сетке приложения.
